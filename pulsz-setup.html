Pulsz Setup

How to Connect

Follow these steps to get your session path and CSRF token from Pulsz and connect to BettrBot.

1

Sign into Pulsz

Go to pulsz.com in your browser and sign into your account.

2

Open Developer Tools

Press F12 on your keyboard (or Ctrl + Shift + I on Windows, Cmd + Option + I on Mac) to open DevTools.

Click the Network tab at the top of the DevTools panel.

3

Open Classic Multihand Blackjack

Navigate to and open Classic Multihand Blackjack. Wait for it to fully load.

4

Find the :path

In the Network tab, click on Fetch/XHR to filter the requests. Look for a request named something like 60b02338-370a-439a-b13f-286417491ac6 (yours will be different, but it will follow the same format).

Click on it, then go to the Headers tab. Scroll down to Request Headers and find the :path header. It will look like:

/api/ClassicMultihandBlackjack/17112523/60b02338-370a-439a-b13f-286417491ac6

Copy the entire :path value — this is your session path.

5

Find the x-csrf-token

In the same request headers, look for (you may need to scroll all the way down):

x-csrf-token: f0anAqiCp97yKJXl...

Copy the entire value — this is your CSRF token.

6

Paste into BettrBot

Open BettrBot, select Pulsz, and click Manual Setup. Paste the :path into the Session Path field and the x-csrf-token into the CSRF Token field, then click Connect.

You're ready to play!

Frequently Asked Questions

I don't see the request in the Network tab ▾
Make sure you have DevTools open with the Network tab and Fetch/XHR selected before you open the Blackjack game. The requests fire when the game loads, so DevTools needs to be recording first. If you missed it, refresh the page.
My session stopped working — what do I do? ▾
Pulsz sessions expire after a period of time. When this happens, click Clear Session in BettrBot, then repeat the steps above to grab a fresh :path and x-csrf-token.
Make sure you're in SC mode ▾
BettrBot plays in Sweepstakes Coins (SC) mode on Pulsz. Make sure you're in SC mode before opening the Blackjack game — the session you capture is tied to whichever coin mode you're in.
Can I keep playing in the browser while BettrBot is running? ▾
No — do not play in the browser at the same time as BettrBot. This can cause conflicts and errors. Close the game in your browser once BettrBot is connected and running.
