ClubsPoker Setup

How to Connect

Follow these steps to grab your Session ID from ClubsPoker and connect to BettrBot.

1

Open ClubsPoker and Log In

Go to clubspoker.com in your browser and log into your account.

2

Open Developer Tools

Press F12 on your keyboard (or Ctrl + Shift + I on Windows, Cmd + Option + I on Mac) to open DevTools.

Click the Network tab at the top of the DevTools panel, then click Fetch/XHR to filter the requests.

3

Open Blackjack

Find and open Blackjack in the ClubsPoker game lobby. The game tile looks like this:

Wait for it to fully load — the requests we need will appear in the Network tab as soon as the game loads.

4

Find the Session ID

In the Network tab, look for a request named blackjack_server.php.

Click on that request, then go to the Payload tab (or Request tab depending on your browser). Find the line labeled session_id — the value will look like:

gth_7635365546318808220

Copy the entire value — that's your Session ID.

5

Paste into BettrBot

Open BettrBot, select ClubsPoker, paste the Session ID into the Session ID field, and click Connect.

Your balance will load and you're ready to play!

Frequently Asked Questions

Where exactly do I find the Session ID in DevTools? ▾
Click the blackjack_server.php request in the Network tab, then look for the Payload tab. The form data will list several fields including session_id, game_id, and username. Copy the value of session_id only — it always starts with gth_ followed by digits.
My session stopped working — what do I do? ▾
ClubsPoker sessions expire after a period of inactivity. When this happens, click Clear Session in BettrBot, then repeat the steps above to grab a fresh Session ID from a new browser session.
I don't see any requests in the Network tab ▾
Make sure you have the Network tab open and Fetch/XHR selected before you open the blackjack game. The requests only appear while DevTools is recording. If it's still empty, try closing and reopening the game with DevTools open.
Which request should I click on? ▾
Look for any request named blackjack_server.php at games-dcw.netgaming.com — that's the NetGaming engine ClubsPoker uses for blackjack. Any request to that endpoint will have the Session ID in its payload.
Do I need to play in SC or GC mode? ▾
BettrBot plays in Sweepstakes Coins (SC) mode on ClubsPoker. Make sure you're playing in SC mode when you capture your Session ID — the session is mode-specific, so a GC session won't work for SC play.
Can I keep playing in the browser while BettrBot is running? ▾
No — do not play in the browser at the same time as BettrBot. This can cause conflicts and errors. Close the game in your browser once BettrBot is connected and running.
