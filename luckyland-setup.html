LuckyLand Slots Setup

How to Connect

Follow these steps to get your Bearer token and Account ID from LuckyLand Slots and connect to BettrBot.

1

Open LuckyLand Slots and Log In

Go to luckylandslots.com in your browser and log into your account.

2

Open Developer Tools

Press F12 on your keyboard (or Ctrl + Shift + I on Windows, Cmd + Option + I on Mac) to open DevTools.

Click the Network tab at the top of the DevTools panel.

3

Open Big Hit Blackjack

Navigate to and open Big Hit Blackjack. Wait for it to fully load.

4

Play One Hand Manually

Place a bet and play one hand of Blackjack in the browser. You only need to make one action (hit, stand, etc.).

5

Find the Bearer Token

In the Network tab, click on Fetch/XHR to filter the requests. Look for a request to games.

Click on it, then go to the Headers tab. Scroll down to Request Headers and find:

authorization: Bearer eyJraWQiOiJhcGktZ2F...

Copy everything after "Bearer " — that entire long string is your token.

6

Find the Account ID

In the same request headers, look for the line that says:

account-id: 12345678

Copy the number — this is your Account ID. It's usually 7-8 digits.

7

Paste into BettrBot

Open BettrBot, select LuckyLand Slots, and click Manual Setup. Paste the Bearer token into the Bearer Token field, paste the Account ID into the Account ID field, and click Connect.

Your balance will load and you're ready to play!

Frequently Asked Questions

Where do I find the Account ID? ▾
The Account ID is in the same request headers where you found the Bearer token. Look for the header labeled account-id — it's a 7-8 digit number. It should be right near the authorization header.
My token stopped working — what do I do? ▾
LuckyLand tokens expire after a period of time. When this happens, click Clear Session in BettrBot, then repeat the steps above to grab a fresh Bearer token and Account ID. The Account ID usually stays the same, but you'll need a new token each time.
I don't see any requests in the Network tab ▾
Make sure you have the Network tab open and Fetch/XHR selected before you play a hand. The requests only appear while DevTools is recording. If it's still empty, try refreshing the page with DevTools open, then play another hand.
Which request should I click on? ▾
Look for requests with names like games. Any of these will have the Bearer token and Account ID in the request headers. If you see multiple, just pick one — they all have the same credentials.
Does BettrBot work with Gold Coins or Sweepstakes Coins? ▾
BettrBot plays in Sweepstakes Coins (SC) mode on LuckyLand. Make sure you're playing in SC mode when you capture your token — the token is the same for both modes, but BettrBot will send bets as SC.
Can I keep playing in the browser while BettrBot is running? ▾
No — do not play in the browser at the same time as BettrBot. This can cause conflicts and errors. Close the game in your browser once BettrBot is connected and running.
